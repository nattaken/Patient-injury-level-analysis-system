# ระบบประเมินระดับความเสี่ยงผู้ป่วย (Triage System)
โปรเจกต์นี้พัฒนาเพื่อฝึกเขียนโปรแกรมภาษา Python ระดับเบื้องต้น  

โดยเป็นระบบจำลองการคัดแยกระดับความเร่งด่วนของผู้ป่วย (Emergency Triage)
# วัตถุประสงค์ของ FilesLock
- ศึกษาเทคนิคการเข้ารหัสและการพัฒนาเว็บไซต์เพื่อสร้างระบบรับส่งไฟล์ที่ปลอดภัย
  
- เพิ่มความปลอดภัยของไฟล์ ลดความเสี่ยงจากการเข้าถึงหรือรั่วไหลของข้อมูล
  
- ส่งเสริมความรู้และความตระหนักด้าน  ความปลอดภัยไซเบอร์แก่ผู้ใช้
# เทคนิคที่ใช้
- ใช้เพียงโครงสร้าง if–else

- ใช้เฉพาะความรู้พื้นฐานของภาษา Python
# หลักการทำงาน
1. ผู้ใช้กรอกข้อมูลต่อไปนี้

- ค่าความดันโลหิต (SBP)

- ชีพจร (HR)

- อาการสำคัญ (Chief Complaint)

2. ระบบจะทำการประเมินผลและจัดระดับความเร่งด่วนของผู้ป่วย  
  → Level 1 - 5 (จากเร่งด่วนสูงสุดถึงน้อยสุด)
